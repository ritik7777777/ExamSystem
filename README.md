# Online Examination System

An ASP.NET MVC based Online Examination System.

## Features
- Admin & Student login
- Exam creation & question management
- Online exam with timer
- Auto evaluation & result generation
- Secure password hashing
- Result history for students

## Tech Stack
- ASP.NET MVC (.NET Framework)
- SQL Server
- ADO.NET
- Bootstrap
- JavaScript
- Git & GitHub

## How to Run
1. Clone the repository
2. Open solution in Visual Studio
3. Create database using SQL scripts
4. Update connection string
5. Run using IIS Express
